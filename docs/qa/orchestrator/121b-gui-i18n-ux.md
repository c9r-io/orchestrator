# QA-121b: GUI i18n / 响应式 / 构建分发

**关联**: FR-069, Design Doc 079
**分类**: GUI / UX
**优先级**: P2

## 场景 1: 入口可见性 — i18n 字符串抽取

**入口**: 启动 GUI 后，顶部导航栏右侧有语言切换按钮

1. 在 `gui/src/` 目录（排除 `lib/i18n.ts`）搜索中文字符
2. 确认搜索结果为零
3. 打开 `gui/src/lib/i18n.ts`，确认所有 UI 文案集中管理

**验证命令**:
```bash
grep -rP '[\x{4e00}-\x{9fa5}]' gui/src/ --include='*.tsx' --include='*.ts' -l | grep -v i18n.ts
# 期望输出为空
```

**预期**: 所有中文字符串集中在 `i18n.ts`，组件文件无硬编码中文。

## 场景 2: 响应式布局

**入口**: 启动 GUI 后，调整窗口大小

1. 调整窗口宽度至 960px（最小值），观察布局不溢出、不破碎
2. 尝试将窗口缩小至 960px 以下，验证 Tauri 阻止进一步缩小
3. 将窗口扩展至 1200px+，观察内容区域适当扩展

**预期**: 960px 最小宽度强制，各断点布局正常。

## 场景 3: 产品构建

1. 执行 `cargo tauri build`
2. 验证 macOS 上生成 `.dmg` 文件
3. 打开 `.dmg`，确认应用可安装运行

**预期**: 构建成功，生成可安装的 macOS .dmg。

## Checklist

- [ ] S1: 语言切换按钮可见，中文字符串全部集中在 i18n.ts
- [ ] S2: 960px 最小宽度强制，响应式断点布局正常
- [ ] S3: 构建生成可安装的 macOS .dmg
